# 🏠 DashboardGPS - Residential Property Finder

A GPS-enabled residential property listing website built with React, TypeScript, and Leaflet maps.

## ✨ Features

- 📍 **GPS Location Detection** - Find properties near your current location
- 🗺️ **Interactive Maps** - Visual property browsing with Leaflet/OpenStreetMap
- 🔍 **Property Search** - Filter by location, price, and amenities
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile
- 👥 **User Roles** - Tenant and landlord dashboards
- 🏷️ **Property Management** - Add, edit, and list properties
